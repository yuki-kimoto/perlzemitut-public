スカラコンテキスト

Perlにはコンテキストと呼ばれる概念があり、スカラとして評価される文脈をスカラコンテキストといいます。

スカラ変数への代入

スカラ変数への代入はスカラコンテキストです。配列はスカラコンテキストで評価すると、配列の長さを返します。 

my $num = @values;

比較演算子の項

比較演算子の左右の項はスカラコンテキストです。 

$x < @values
$x == @values

for文のループで頻繁に目にします。 

for (my $i = 0; $i < @nums; $i++) {
  ...
}

ifの条件部

ifの条件部はスカラコンテキストです。以下では@valuesはスカラコンテキストで評価されるので、配列の個数を返します。 

if (@values) {
  ...
}
