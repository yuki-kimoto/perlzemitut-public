Perlリリース情報 - バージョンごとの機能追加を紹介

Perlのリリース記事をこちらにまとめておきます。Perlのバージョン間の変更点などを把握するのにお役立てください。Perl 5.8.7で動くように記述しておくと、ほぼすべての環境で実行できます。

Perl 5.28

  • Perl 5.28リリース - Unicode 10.0サポート、文字列連結の高速化、ref関数、keys関数、forループの高速化

Perl 5.26

  • Perlの最新バージョンPerl 5.26リリース。インデント可能ヒアドキュメント、Unicode 9.0サポート、長いキーのハッシュのパフォーマンスの改善

Perl 5.24

  • Perlの最新バージョン 5.24リリース - サブルーチンと数値計算の高速化、Unicode 8.0のサポート、文法機能強化

Perl 5.22

  • Perl 5.22がリリースされました - 数学関連の機能強化、メソッドのパフォーマンスの改善、正規表現の改善 (2015年)

Perl 5.20

  • Perl 5.20がリリース - ipv6のサポート、サブルーチンの新しい文法、新しいスライス記法、Unicode 6.3への対応など (2014年)

Perl 5.18

  • Perl 5.18がリリースされました (2013年)

Perl 5.16

  • Perlの最新バージョン5.16.0がリリースされました (2012年)

Perl 5.14

  • Perl 5.14 リリース (2011年)

Perl 5.6

  • ourによるパッケージ変数の宣言

Perl 5.0

  • myによるレキシカル変数の宣言

Perl 1.0

  • 1988年Perl 1.0公開
