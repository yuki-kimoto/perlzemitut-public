繰り返し演算子「x」

繰り返し演算子「x」を使用すると、文字列あるいは配列を繰り返し回数だけ、連結することができます。 

# 文字列を繰り返し連結
my $str2 = $str1 x 3;

# 配列を繰り返し連結
my @array2 = @array1 x 3;

文字列を繰り返し連結するサンプルプログラムです。 

my $str = 'abc' x 3;

$strは「abcabcabc」になります。

配列を繰り返し連結するサンプルです。 

my @nums = (1, 2, 3) x 3;

@numsは(1, 2, 3, 1, 2, 3, 1, 2, 3)になります。
