ディレクトリの存在を確認する

ディレクトリが存在するかどうかを確認するには、ファイルテスト演算子「-d」を使用します。 

-d $file

ディレクトリが存在すると真値を、存在しないと偽値を返します。

ファイル演算子 -d を使ってディレクトリの存在を確認するサンプルです。 

use strict;
use warnings;

# ディレクトリの存在を確認する。
# -d ディレクトリ名

print "1: ディレクトリの存在を確認する。 -d \n";
my $dir = 'd';
if (-d $dir) { 
  print "'$dir' は、存在します。\n";
}
else { print "'$dir' は、存在しません。\n" }
