ファイル名を指定してモジュールを読み込む

ファイル名を指定してモジュールを読み込むには、requireの引数にファイル名を文字列で渡します。 

require ファイル名;
カレントディレクトリのSample.pmが読み込まれます。 
use strict;
use warnings;

require "Sample.pm";

# 1 が出力される
print one();

Sample.pm

sub one { return 1 }
1;
