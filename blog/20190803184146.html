Giblog 1.0 リリース - Gitで管理できるWebサイトとブログ生成ツール

Giblog 1.0 リリース。Giblogは、Perlゼミ製作のGitで管理できるWebサイトとブログ生成ツールを生成するためのツールです。

Giblog公式サイト - Gitで管理できるWebサイト・ブログの作成のツール

Gitで管理できる静的なWebサイトを簡単に作成できます。

  • すべてのページをGit管理
  • 高度なSEO対策
  • レスポンシブ対応
  • お問い合わせフォーム
  • 高速な再構築
  • 高度なセキュリティ
  • 記事の検索・置換が簡単
  • Windows Mac Linux対応
  • Perlで自由にカスタマイズ

645ページを0.78秒で再構築。ページのすべてを、静的なファイルで管理できるから、Perlを使った編集・置換も自由自在。

すべてが静的なページなので、ページ書き換えなどの脆弱性を生み出さない最強のセキュリティ。

Windows、Mac、Linux対応。

Twitterカードを追加したいなどの機能要望は、Perlを使って、実現できる。

Github Pagesにアップロードすれば、あなたのページが、すぐにできちゃう。

CGIを利用できる環境であれば、お問合せフォームもサポート。

Perlゼミで実証されている、高度なSEO対策をデフォルトで提供。

デフォルトでスマートフォン対応。現代的なWebサイトデザインもCSSの修正で対応可能です。

Giblog公式サイトはもちろんGiblogで製作されていますよ!
